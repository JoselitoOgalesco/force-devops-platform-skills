# sf-diagram

Create architecture and flow diagrams for Salesforce solutions.

## Description

This skill helps you create professional Mermaid diagrams for Salesforce architectures, data models, integration flows, and system designs. Covers ERD diagrams, sequence diagrams, flowcharts, class diagrams, and deployment diagrams with Salesforce-specific patterns.

## Features

- **ERD Diagrams** — Object relationships and data models
- **Sequence Diagrams** — Integration flows and API calls
- **Flowcharts** — Process flows and decision trees
- **Class Diagrams** — Apex class hierarchies
- **Deployment Diagrams** — System architecture views
- **Salesforce Conventions** — Standard vs custom object notation

## Quick Start

1. Identify diagram type (ERD, sequence, flowchart)
2. List entities/components to include
3. Define relationships/flows
4. Generate Mermaid syntax
5. Render and refine

## Usage

Invoke this skill when:
- Documenting data models
- Designing integration architectures
- Explaining complex processes
- Creating technical documentation
