# sf-schema

Design and manage Salesforce object schemas.

## Description

This skill provides guidance for designing custom objects, fields, relationships, and data models. Covers naming conventions, field types, relationship patterns, and metadata best practices.

## Features

- **Object Design** — Custom objects with proper configuration
- **Field Types** — Choose appropriate field types
- **Relationships** — Lookup vs Master-Detail patterns
- **Naming Conventions** — API names and labels
- **Metadata Structure** — XML format and deployment
- **Indexes** — Custom indexes for performance

## Quick Start

1. Define object purpose and relationships
2. Choose appropriate field types
3. Set naming conventions (Object__c, Field__c)
4. Configure page layouts and record types
5. Create relationships with proper deletion behavior

## Usage

Invoke this skill when:
- Designing new custom objects
- Adding fields to existing objects
- Planning data model relationships
- Reviewing metadata structure
