# sf-permissions

Configure Salesforce permissions and access control.

## Description

This skill provides comprehensive guidance for Permission Sets, Permission Set Groups, Profiles, Sharing Rules, and Role Hierarchy. Covers the shift from Profiles to Permission Sets, muting permissions, and automated permission assignment.

## Features

- **Permission Sets** — Modular permission assignment
- **Permission Set Groups** — Bundle related permission sets
- **Muting Permission Sets** — Override permissions in groups
- **Profile Migration** — Moving from Profiles to Permission Sets
- **Sharing Rules** — OWD and criteria-based sharing
- **Role Hierarchy** — Record access inheritance

## Quick Start

1. Audit current Profile permissions
2. Create granular Permission Sets by function
3. Build Permission Set Groups for personas
4. Assign via Flows or Apex triggers
5. Test with different user contexts

## Usage

Invoke this skill when:
- Designing permission architecture
- Migrating from Profiles to Permission Sets
- Implementing sharing rules
- Automating permission assignment
