# sf-docs

Generate and maintain Salesforce documentation.

## Description

This skill provides guidance for creating comprehensive Salesforce documentation including ApexDoc comments, README files, runbooks, and technical specifications. Covers documentation standards, code comments, and knowledge base articles.

## Features

- **ApexDoc Comments** — Standardized Apex documentation
- **README Templates** — Project and component documentation
- **Runbooks** — Operational procedures and troubleshooting guides
- **Technical Specs** — Design documents and architecture decisions
- **Release Notes** — Change documentation for deployments
- **Knowledge Base** — End-user documentation

## Quick Start

1. Add ApexDoc to all public methods and classes
2. Create README for each major component
3. Document deployment procedures
4. Maintain change log

## Usage

Invoke this skill when:
- Writing Apex class documentation
- Creating project README files
- Documenting deployment procedures
- Writing technical specifications
