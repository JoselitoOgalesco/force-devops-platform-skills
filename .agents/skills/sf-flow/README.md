# sf-flow

Build and optimize Salesforce Flows.

## Description

This skill provides guidance for creating Screen Flows, Record-Triggered Flows, Scheduled Flows, and Autolaunched Flows. Covers flow best practices, bulkification, error handling, and flow testing strategies.

## Features

- **Flow Types** — Screen, Record-Triggered, Scheduled, Autolaunched
- **Bulkification** — Handle multiple records efficiently
- **Error Handling** — Fault paths and error notifications
- **Flow Testing** — Debug mode and test coverage
- **Flow Optimization** — Reduce elements, optimize queries
- **Flow Triggers** — Before-save vs after-save patterns

## Quick Start

1. Choose appropriate flow type for use case
2. Design with bulkification in mind
3. Add fault paths for error handling
4. Test with multiple records
5. Activate and monitor

## Usage

Invoke this skill when:
- Building new flows
- Optimizing existing flows
- Converting Process Builder to Flow
- Implementing record-triggered automation
