# sf-lwc

Scaffold Lightning Web Components with best practices.

## Description

This skill helps you create production-ready Lightning Web Components including HTML templates, JavaScript controllers, CSS styles, meta.xml configuration, and Jest tests. Covers reactive properties, wire adapters, imperatively called Apex, and component communication.

## Features

- **Component Scaffolding** — Complete bundle structure
- **Wire Adapters** — Reactive data binding
- **Apex Integration** — @wire and imperative calls
- **Component Communication** — Events and public properties
- **Jest Testing** — Unit tests for LWC
- **SLDS Styling** — Lightning Design System compliance

## Quick Start

1. Create component folder (camelCase name)
2. Add HTML, JS, CSS, and meta.xml files
3. Configure targets in meta.xml
4. Add Jest tests in __tests__ folder
5. Deploy and test in org

## Usage

Invoke this skill when:
- Creating new LWC components
- Adding Apex integration to LWC
- Writing Jest tests for LWC
- Implementing component communication
