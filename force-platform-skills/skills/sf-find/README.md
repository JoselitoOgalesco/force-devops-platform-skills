# sf-find

Search and locate Salesforce metadata and code.

## Description

This skill helps you find Apex classes, triggers, LWC components, flows, and other metadata in a Salesforce project or org. Covers local file search, org metadata queries, and dependency analysis.

## Features

- **Local Search** — Find files in SFDX project structure
- **Org Metadata Query** — Tooling API queries for metadata
- **Dependency Analysis** — Where-used and what-uses queries
- **Code Search** — Grep patterns for Apex, LWC, Flows
- **Component Lookup** — Find components by name or type

## Quick Start

1. Identify what you're looking for (type, name pattern)
2. Use appropriate search (local vs org)
3. Review results and filter
4. Navigate to relevant files

## Usage

Invoke this skill when:
- Locating specific Apex classes or triggers
- Finding where a field is used
- Searching for components by name
- Analyzing dependencies
