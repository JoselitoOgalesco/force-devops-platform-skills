# sf-omnistudio

Build OmniStudio components and integrations.

## Description

This skill provides guidance for building OmniStudio solutions including OmniScripts, FlexCards, DataRaptors, and Integration Procedures. Covers component design, data mapping, and deployment strategies.

## Features

- **OmniScripts** — Guided digital experiences
- **FlexCards** — Contextual information display
- **DataRaptors** — Data extraction and transformation
- **Integration Procedures** — Complex data orchestration
- **LWC Integration** — Custom LWC in OmniStudio
- **Deployment** — OmniStudio metadata migration

## Quick Start

1. Design the user journey (OmniScript)
2. Create DataRaptors for data operations
3. Build FlexCards for information display
4. Use Integration Procedures for complex logic
5. Test and deploy

## Usage

Invoke this skill when:
- Building guided user experiences
- Creating data transformation logic
- Designing contextual UI components
- Migrating OmniStudio metadata
